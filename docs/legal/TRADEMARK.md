## 2. 許可される利用 (Permitted Uses)

以下の利用は事前許諾なしに行えます。ただし混同を招かない形式に限ります。

- **事実に基づく参照 (fact-based reference)**
  - 「本プロジェクトは `ai-chan` と相互運用する」等の事実記述。
- **学術引用 (academic citation)**
  - 論文・技術ブログ・レポートでの名称引用。引用元を明記してください。
- **相互運用 (interoperability)**
  - API や設定ファイル名として名称に言及すること。
  - 「`hinomoto-model` 互換レイヤー」等の機能説明。
- **比較・批評 (comparative / critical commentary)**
  - 公正な比較や批評。誤解を招く形式でないこと。

以下は明示的に禁止します。違反が確認された場合、修正要請・配布停止要請を行います。

- **派生モデル名への冠用**

    派生モデル / 派生プロダクトの公開配布。

  - 「ai-chan 公式」「HinoMoto 認定」など、本プロジェクトによる公認を示唆する表現。
- **ロゴ・アイコン・立ち絵 (AiChan.app 内アセット等) の無断転用**
