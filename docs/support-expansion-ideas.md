# サポート機能拡張アイデア

アイを「会話相手」から「パーソナルアシスタント」に進化させるためのアイデア集。

---

## 実装済み（今回追加）

### ドキュメント出力
- Word / PowerPoint / Excel への出力
- 「Wordにまとめて：〇〇」「パワポで資料作って：〇〇」「エクセルに一覧作って：〇〇」
- LLMが内容を構造化 → Office形式に変換

---

## 短期（すぐ実装可能）

### 1. リサーチアシスタント
**概要**: Webから情報を集めて整理し、ドキュメントにまとめる一連のフロー
**コマンド例**:
- 「〇〇について調べてまとめて」→ Web検索 → 要約 → Word出力
- 「〇〇と△△を比較して表にして」→ 調査 → 比較表 → Excel出力

**必要な追加**:
- web_fetcher の検索機能強化（複数ソース自動巡回）
- 調査→構造化→出力のパイプライン化

### 2. 会議サポート強化
**概要**: 議事録だけでなく、事前準備〜事後フォローまで
**コマンド例**:
- 「〇〇の会議のアジェンダ作って」→ PowerPoint出力
- 「議事録をWordにして」→ 議事録→Word変換
- 「先週の会議のアクションアイテム確認して」→ 進捗追跡

### 3. メール・文書の下書き
**概要**: ビジネスメール、報告書、提案書のドラフト作成
**コマンド例**:
- 「〇〇さんへのメール書いて：内容〇〇」
- 「週次報告書作って」→ タスク完了状況からWord自動生成
- 「提案書のテンプレート作って」→ PowerPoint出力

### 4. 日報・週報の自動生成
**概要**: その日/週の会話履歴・タスク完了状況から自動で日報作成
**コマンド例**:
- 「今日の日報作って」→ 会話ログ + タスク完了 → Word出力
- 「今週のサマリー」→ 週間の活動まとめ → Excel/Word

---

## 中期（開発が必要）

### 5. データ分析アシスタント
**概要**: CSVやExcelを読み込んで分析・可視化
**コマンド例**:
- 「このCSV分析して：/path/to/data.csv」
- 「売上データをグラフにして」→ matplotlibで可視化
- 「データのサマリーをExcelにして」

**必要な追加**:
- pandas でのデータ読み込み・分析
- matplotlib/plotly でのグラフ生成
- Plan E（サンドボックス）と連携

### 6. 翻訳アシスタント
**概要**: ドキュメントの翻訳、多言語対応
**コマンド例**:
- 「この文章を英語に翻訳して」
- 「この英語のPDF読んで日本語でまとめて」

### 7. スケジュール管理の強化
**概要**: Google Calendar連携 + タスク管理を統合
**コマンド例**:
- 「今週の空き時間教えて」
- 「〇〇さんとの会議をセットして」
- 「締め切りが近いタスク教えて」

### 8. プロジェクト管理
**概要**: 複数プロジェクトのタスク・進捗を管理
**コマンド例**:
- 「プロジェクト一覧」
- 「プロジェクトAの進捗を報告書にして」→ Word出力
- 「今月のマイルストーン確認」

## 長期（成長段階に応じて）

### 9. 開発サポート（Plan E連携）
**概要**: コードの生成・分析・テスト実行
**前提**: 成長段階が青年期 + サンドボックス実装後
**コマンド例**:
- 「この関数のテスト書いて」
- 「このエラー調べて直して」
- 「APIの仕様書作って」→ Word出力

### 10. マルチモーダルレポート
**概要**: 画像・グラフ・表を含むリッチなレポート生成
**コマンド例**:
- 「スクリーンショット付きで手順書作って」
- 「データ分析結果をパワポにまとめて」（グラフ付き）

```
短期（1-2日で実装可能）:
  ① リサーチ→ドキュメント出力パイプライン
  ② 議事録のOffice出力
  ③ メール・文書テンプレート
  ④ 日報・週報自動生成

中期（1-2週間）:
  ⑤ データ分析（pandas + グラフ）
  ⑥ 翻訳アシスタント
  ⑦ スケジュール管理強化
  ⑧ プロジェクト管理

長期（成長と共に）:
  ⑨ 開発サポート
  ⑩ マルチモーダルレポート
```

## 共通設計方針
- **全てOffice出力可能に**: 調べた結果、分析結果、議事録 → Word/PPT/Excelで出力
- **パイプライン思考**: 入力→処理→構造化→出力を一貫して
- **成長と連動**: 機能は成長段階に応じて解放
- **既存機能の組み合わせ**: 新機能は既存モジュールの組み合わせで実現
