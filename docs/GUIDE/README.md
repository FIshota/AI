## 10 年後の最上位ゴール（2026-04-20 確定）

> **Ai が家族として量産される社会を実現する**
> （TryEarth / HINOMOTO プロジェクト）
>
> **三系統**で実現：
> - **Ai**（プロトタイプ・創業者個人のパートナー・非公開）
> - **YAMATO**（国家モデル・国産 AI として国の基幹システム支援・MIT OSS）
> - **KAGUYA**（一般家庭モデル・量産・デュアルライセンス）

詳細は `06-three-model-strategy.md`。

## 譲れない一点（モデル別・2026-04-20 確定）

| モデル | 主軸 | 補助 |
|:-:|:-:|:-:|
| **Ai** | 🅲 人格の連続性 | A, D |
| **YAMATO** | 🅰 ローカルファースト | C, D |
| **KAGUYA** | 🅳 優しさ・正直さ + 🅴 ゼロ円運用 | A |

**三つ共通の最低ライン**: A（ローカルファースト）+ D（優しさ正直さ）は全モデル必須。

凡例：
- 🅰 ローカルファースト（クラウド送信ゼロ）
- 🅱 MIT OSS
- 🅲 人格の連続性
- 🅳 家族への優しさ・正直さ（4 軸評価）
- 🅴 ゼロ円運用可能
