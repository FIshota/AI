## 🚨 Critical / High Findings

### [HIGH] CVE-2025-69872 — diskcache 5.6.3

| Field | Detail |
|---|---|
| **CVE** | CVE-2025-69872 |
| **GHSA** | GHSA-w8v5-vhqr-4h9v |
| **Package** | diskcache 5.6.3 |
| **Fix Version** | **No fix available** |
| **Impact** | Arbitrary Code Execution (RCE) via Python `pickle` deserialization |

**Description:** DiskCache (python-diskcache) through 5.6.3 uses Python `pickle` for serialization by default. An attacker with **write access to the cache directory** can achieve arbitrary code execution when a victim application reads from the cache.

**Mitigation (until a fix is released):**
- Restrict filesystem permissions on the cache directory to trusted processes only.
- Evaluate replacing `pickle` with a safe serializer (e.g., `msgpack`, `json`) via DiskCache's `disk` parameter.
- Monitor GHSA-w8v5-vhqr-4h9v for upstream patch availability.
