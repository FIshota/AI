3. **[MEDIUM] Audit `server_home.py:265` Paramiko `exec_command(cmd)`** — The `cmd` variable passed to SSH `exec_command` must be validated against an explicit allowlist. If arbitrary command execution is required, document the trust boundary and add input sanitisation.

5. **[LOW] Upgrade `pip` / `setuptools` / `packaging`** — These are infrastructure dependencies. Upgrading pip (24→26) and setuptools (68→82) reduces exposure to supply-chain risks and improves resolver security.
