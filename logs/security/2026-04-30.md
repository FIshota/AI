## Recommendations (優先度順)

1. **[HIGH] Monitor diskcache CVE-2025-69872** — No fix version is available yet. Consider replacing `diskcache` with an alternative (e.g. `cachetools`) or adding a runtime guard until a patch is released. Track the upstream issue.

2. **[MEDIUM] Upgrade `cryptography` 41 → 47** — This is a security-sensitive library spanning 6 major versions behind. Breaking changes exist; review the [changelog](https://cryptography.io/en/latest/changelog/) before upgrading.

3. **[MEDIUM] Audit Paramiko `exec_command` inputs in `core/server_home.py`** — Six call sites (lines 241, 265, 298, 302, 306, 312) pass command strings. Verify that any user-controlled data is sanitised or that commands are fully hardcoded before reaching `exec_command`.

4. **[MEDIUM] Review SQL f-string construction in `core/conversation_search.py`** — Lines 306 and 368 build SQL with f-strings. Confirm that column/table name variables come from a fixed allow-list, not user input, or migrate to an ORM / parameterised query builder.

5. **[LOW] Batch-upgrade remaining 23 non-major outdated packages** — `certifi`, `PyJWT`, `setuptools`, `packaging`, and `pip` have minor/patch updates with security relevance. Run `pip install --upgrade <pkg>` in a staging environment.
