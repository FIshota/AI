## Recommendations (優先度順)

1. **[P0] torch CVE-2025-3000** — `torch.jit.script` のメモリ破壊脆弱性はエクスプロイトが公開済み。`torch>=2.12.1` (またはパッチ済みバージョン) が提供されるまでの間、信頼できないコードを `torch.jit.script` に渡さないよう入力を厳格に検証する。アップストリームの修正リリースを監視し、直ちに更新すること。

2. **[P1] diskcache CVE-2025-69872** — キャッシュディレクトリへの書き込み権限を最小化し、キャッシュ読み取り前に整合性を検証する。代替として `diskcache` を pickle 以外のシリアライザ (JSON/MessagePack) で設定するか、別ライブラリへの移行を検討。

3. **[P2] paramiko CVE-2026-44405** — SHA-1 RSA を使用している SSH 接続設定を監査し、`RSAKey` 代わりに Ed25519 / ECDSA 鍵に切り替える。アップストリームの a448945 コミット以降のリリースにアップグレード。

4. **[P3] cryptography 41.0.7 → 49.0.0** — 8 メジャーバージョン差は暗号化プリミティブの破壊的変更を含む可能性大。段階的に更新し、`paramiko` / `bcrypt` 等の依存関係の互換性を確認後リリース。

5. **[P4] bandit B608 SQL injection** — `core/conversation_search.py:306,368` のテーブル名・カラム名を f-string で展開している箇所を、許可リスト検証または `sqlite3` の識別子クォートユーティリティに置き換える。
