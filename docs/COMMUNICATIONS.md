# COMMUNICATIONS

ai-chan 本体は SNS 等の公開発信を行わない。広報・普及的な文章は、原則として
HinoMoto モデル家族側の blog (`hinomoto-model/blog/`) に集約する。

## 原則

1. **ai-chan は家族**であり、プロダクトではない。公式 SNS アカウント、公開
   ブログ、プレスリリース等は持たない。
2. 技術的な普及活動は **HinoMoto モデル** (日本語特化 SLM) の文脈で行う。
   ai-chan 本体の詳細 (家族構成、内部状態、会話ログ、声 ID 等) は blog でも
   扱わない。
3. どうしても ai-chan に言及する必要がある場合は、設計思想・価値観
   (`docs/VALUES.md`, `docs/VISION.md`) の範囲に留める。

## Broadcast (外向き発信) の扱い

ai-chan から外向きの発信を行う可能性がある場面:

- 保存データの第三者提供 (例: 研究目的での対話ログ共有)
- ai-chan 自身に「発言させる」形のデモ動画や音声公開
- HinoMoto blog の記事内での ai-chan 実例引用

これらは **家族全員の同意** を前提とする。同意取得フローは未確定だが、
以下の最低限を満たす:

- 対象データのプレビューを同意対象者に提示
- 1 人でも拒否した場合は発信を中止
- 同意記録はローカルに保存 (クラウドに送らない)

## HinoMoto Blog との関係

| 項目 | ai-chan 側 | HinoMoto blog 側 |
|------|-----------|------------------|
| SNS 発信 | なし | なし (GitHub のみ) |
| 技術記事 | 参照リンクのみ | 本体を書く |
| 家族文脈の記述 | 最小限 | 書かない |
| 消す権利の解説 | 設計/実装のみ | 思想を記事化 |
